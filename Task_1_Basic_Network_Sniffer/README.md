# 🚀 CodeAlpha - Basic Network Sniffer v2.0 (Python)

## 📌 Overview
This project is a **Professional CLI-Based Network Packet Sniffer** built using **Python** and the **Scapy** library.  
It captures live network packets and displays important packet details such as source/destination IPs, protocols, ports, payload preview, and TCP flags.

This project was developed as part of the **CodeAlpha Cybersecurity Internship Program (Task 1)**.

---

## 🎯 Key Features
✅ Interface Selection (eth0 / wlan0 / lo etc.)  
✅ Protocol Filter (TCP / UDP / ICMP / ALL)  
✅ IP Filtering (Capture traffic from a specific IP)  
✅ Port Filtering (Capture traffic for a specific port)  
✅ Summary Mode (Fast output for monitoring)  
✅ Detailed Mode (Deep packet analysis with payload + flags)  
✅ Optional Logging (Save output only if user wants)  
✅ Packet Count Limit (Capture limited packets or unlimited)  
✅ Capture Statistics Report (Protocols + top IPs + top ports)
