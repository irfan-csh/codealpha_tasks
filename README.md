## ✅ Task 3: Network Intrusion Detection & Automated Response System

A Network Intrusion Detection System (NIDS) built using Suricata on Ubuntu Linux to monitor network traffic, detect suspicious activities, and automate incident response.

**Features:**

* Real-Time Network Traffic Monitoring
* Custom Suricata Detection Rules
* ICMP Ping Detection
* SYN Port Scan Detection
* SSH Connection Monitoring
* HTTP Traffic Detection
* Telnet Connection Detection
* Live Alert Monitoring
* Automated IP Blocking using iptables
* IDS to IPS-like Functionality

**Tools Used:**

* Ubuntu Linux
* Suricata IDS
* Kali Linux
* Nmap
* Bash Scripting
* iptables
* VirtualBox

**Project Components:**

* Detailed Project Report (PDF & DOCX)
* Custom Detection Rules
* Attack Simulation Screenshots
* Automated Response Script
* Alert Monitoring Logs

📂 Folder: `CodeAlpha_Network_IDS/`
